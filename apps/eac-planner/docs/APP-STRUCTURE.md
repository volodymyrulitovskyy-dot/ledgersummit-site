# Application Structure

## Top navigation

Recommended primary navigation:

- Dashboard
- Forecast Review
- Labor
- Sub
- Equipment
- Material
- ODC
- Resource Management
- Actuals
- Admin

## Screen responsibilities

### Dashboard

Executive review and forecast health.

### Forecast Review

Monthly review workflow for:

- closing actual period
- comparing working forecast to prior forecast
- reviewing variance explanations
- submitting or approving the new forecast version

### Labor

Employee-level planning with:

- employee dropdown
- labor category dropdown
- organization and department context
- WBS or cost code assignment
- monthly hours entry
- calculated cost and revenue

### Sub

Subcontractor line planning with:

- vendor dropdown
- contract or PO reference
- WBS or cost code
- monthly forecast
- commitments visibility

### Equipment

Equipment line planning with:

- equipment class or asset dropdown
- rate basis
- monthly units or hours
- utilization indicator where relevant

### Material

Material planning with:

- item dropdown
- unit of measure
- unit cost
- monthly quantities

### ODC

Non-labor direct cost planning with:

- ODC type
- line description
- monthly dollars

### Resource Management

Capacity and utilization views using labor planning as the source.

### Actuals

QuickBooks import management and reconciliation.

### Admin

Master data and setup:

- employees
- organizations
- departments
- labor categories
- vendors
- equipment catalog
- material catalog
- WBS
- cost codes
- rate tables

## Navigation behavior

- Use top-level navigation for major work areas
- Use contextual secondary filters inside each screen
- Preserve selected project, forecast version, and period across screens
